# VueTracker

VueTracker - це веб-додаток для управління проєктами та завданнями. Дозволяє створювати проєкти та таски, редагувати їх, видаляти, сортувати та перетягувати (drag & drop). Додатково доступна навігація по проєктах і графічне відображення даних.

---

## Функціонал

- **Проєкти**
  - Створення нового проєкту
  - Редагування проєкту
  - Видалення проєкту
  - Відображення кількості завдань у проєкті
- **Завдання (Tasks)**
  - Створення завдання для конкретного проєкту
  - Редагування завдання
  - Видалення завдання
  - Статус завдання: To Do, In Progress, Done
  - Дедлайн для завдання
- **Інші можливості**
  - Drag & drop для сортування завдань
  - Навігація між проєктами
  - Графік або статистика по завданням 

---

## API ендпоінти

### Проєкти
| Метод | URL | Опис |
|-------|-----|-----|
| POST  | `/projects/create` | Створити новий проєкт |
| PUT   | `/projects/update/:id` | Оновити проєкт за ID |
| DELETE| `/projects/delete/:id` | Видалити проєкт за ID |

### Завдання (Tasks)
| Метод | URL | Опис |
|-------|-----|-----|
| POST  | `/tasks/create` | Створити нове завдання |
| PUT   | `/tasks/update/:id` | Оновити завдання за ID |
| DELETE| `/tasks/delete/:id` | Видалити завдання за ID |

## Інструкція по запуску

1. Клонувати репозиторій:

```bash
git clone https://github.com/vlad113chepurko/ProjecTracker  
cd ProjecTracker  
npm run dev
```

2. Встановити залежності

```bash
npm install
```

3. Запустити фронтенд (Vue 3 + Vite):

```bash
npm run dev
```

4. Запустити бекенд (Node.js сервер):

```bash
cd backend
node server.js
```

## Як працювати з додатком

### Створення проєкту
1. Натиснути **"Add New Project"**
2. Ввести назву, опис і статус
3. Натиснути **"Add Project"**

### Управління завданнями
1. Вибрати проєкт, щоб побачити його завдання
2. Натиснути **"Add Task"** для створення нового завдання
3. Вказати ім’я, автора, статус та дедлайн
4. Для редагування або видалення — використовувати кнопки **"Edit"** та **"Delete"**

### Drag & Drop
- Таски можна перетягувати, змінюючи їх порядок у таблиці

### Фільтри та сортування
- Можна сортувати проєкти та завдання за:
  - статусом
  - іменем
  - ID
  - кількістю завдань

### Навігація
- Натискаючи на рядок проєкту, ви переходите до його завдань

### Технології
- **Vue 3 + Vite** — фронтенд
- **Pinia** — керування станом
- **Vee-Validate** — валідація форм
- **Axios** — HTTP запити
- **Node.js** — бекенд
- **Draggable** — drag & drop для тасок
